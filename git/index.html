Команди Git - допоможе в роботі

git status - Перераховує всі нові або змінені файли, які потребують фіксації

git diff - Показує відмінності щодо внесених змін у ще не проіндексованих файлах

git add - Индексирует указанный файл для последующего коммита

git commit -m "[що змінили]" - Фіксує проіндексовані зміни та зберігає їх в історію версій

git pull - Завантажує історію з віддаленого репозиторію та поєднує її з локальною. pull = fetch + merge

git push original [гілка] -Завантажує всі зміни локальної гілки у віддалений репозиторій
